# Smart Paste
Paste JSON/XML content to XML/JSON or Classes (in progress). Inspired by Visual Studio Paste Special.

## Install
Press `F1`, type `ext install smartpaste`.

## How to
Invoke command pallate, type `smartpaste`, hit enter!

## Features
* Convert JSON to XML
* Convert XML to JSON
